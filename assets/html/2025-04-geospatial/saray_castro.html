1. Introduccin

Los pinginos son un grupo de aves no voladoras que pertenecen al Orden Sphenisciformes, en este orden podemos encontrar una sola familia: Spheniscidae. Existen 18 especies de pinginos en el mundo, y se encuentran distribuidas en el hemisferio Sur, cuatro de estas anidan en el Continente Antrtico y slo una en el rea ecuatorial.

Los pinginos se encuentran especficamente en: Antrtica, Nueva Zelanda, al sur de Australia, Sudfrica, Sur Amrica, Norte del Per y en las islas Galpagos. Su hbitat es marino, utilizando ecosistemas terrestres solo para los distintos ciclos de reproduccin.

Los pinginos son especies indicadoras, ya que el estado de sus poblaciones refleja el estado de salud de los ocanos que habitan. Las condiciones ocenicas estn relacionadas al xito reproductivo del Pingino de Galpagos.

El estudio de los pinginos y conservacin es fundamental, ya que alrededor de dos tercios de las especies de pinginos estn incluidas en la Lista Roja de la UICN de las Especies Amenazadas. (Banda Cruz, Aura Gabriela. 2017).

Se define la especie "Spheniscus demersus" por tener una poblacin suficiente para el anlisis de esta actividad. Inicialmente procedemos identificando la ocurrencia de avistamientos de la especie "Spheniscus demersus".

In[]:
#Para la presente investigacin se selecciona la especie Pinguinos.
especie = "Spheniscus demersus"

limite = 300
offset = 0
registros_acumulados = []

while True:
    resproy = occurrences.search(
        scientificName=especie,
        hasCoordinate=True,
        hasGeospatialIssue=False,
        limit=limite,
        offset=offset
    )
    # Se extraen los resultados
    registros = resproy.get("results", [])

    # Si ya no hay resultados, se detiene el ciclo
    if not registros:
        break

    # Agregar registros nuevos al acumulado
    registros_acumulados.extend(registros)

    #Actualizar el offset para la siguiente "pgina"
    offset += limite
#Convertir todo a un DataFrame de pandas
presencia = pd.DataFrame(registros_acumulados)

import requests # biblioteca para consultar API
import json # biblioteca para manejar datos JSON

# Nombre cientfico de la especie que se desea buscar
nombre_cientifico = 'Spheniscus demersus'

# URL
url = 'https://api.gbif.org/v1/species/match'

# Parmetros de la solicitud
parametros = {
    'name': nombre_cientifico,
    'strict': True  # Asegura una coincidencia exacta en la bsqueda
}

# Solicitud (el resultao se almacena en el objeto respuesta)
respuesta = requests.get(url, params=parametros)

# Verificacin de la respuesta
if respuesta.status_code == 200:
    # La solicitud fue exitosa

    # Se cargan en formato JSON los datos retornados
    datos_proy = respuesta.json()

    # Obtencin del taxonKey de la especie
    taxon_key = datos_proy.get('usageKey', 'No encontrado')

    # Despliegue del taxonKey
    print(f'taxonKey de {nombre_cientifico}: {taxon_key}')

    # Almacenamiento de datos en un archivo llamado "datos-proy.json"
    with open('datos-proy.json', 'w', encoding='utf-8') as archivo:
        json.dump(datos_proy, archivo, indent=4, ensure_ascii=False)
else:
    # Se produjo un error
    print(f"Error {respuesta.status_code}")

import folium

# Mapa centrado en la latitud 0 y longitud -80,
# con un zoom inicial de nivel 2
mapa = folium.Map(location=[20, 40], zoom_start=2)

# URL base de los tiles de GBIF para la especie Spheniscidae (taxonKey=5284)
# En este caso, se usa 'bin=square' para mostrar las celdas cuadradas de densidad
tiles_url = (
    "https://api.gbif.org/v2/map/occurrence/density/"
    "{z}/{x}/{y}@1x.png"
    "?taxonKey=5229384&bin=square&style=green.poly"
)

# Se aade la capa de tiles al mapa Folium
folium.TileLayer(
    tiles=tiles_url,
    attr=" GBIF",            # Atribucin recomendada
    name="Spheniscidae (cuadrculas)",
    overlay=True,
    control=True,
    # max_zoom=14 si se desea limitar el zoom mximo
).add_to(mapa)

# Opcional: se puede aadir otra capa con bin=hex o diferente estilo

    "https://api.gbif.org/v2/map/occurrence/density/"

)
